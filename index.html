🤖 个人AI分身 - API文档

基于Kimi 2.5 + JSON知识库的个人AI分身系统

📋 项目概览

项目名称：个人AI分身（Personal AI Clone）

核心功能：通过System Prompt + Few-Shot + RAG知识库，训练一个具有用户个人色彩的AI分身

最终形态：微信小程序 + API接口（可出租给第三方平台）

训练周期：3个月（数据积累 → 模式提取 → 产品化）

🛠️ 技术栈

LLM模型
Kimi 2.5（阿里云百炼）
后端框架
Node.js + Express
知识库
JSON格式（本地存储）
前端
微信小程序

🔌 API接口

1. 对话接口

POST /api/v1/chat
POST /api/v1/chat Content-Type: application/json Authorization: Bearer YOUR_API_KEY { "message": "用户输入的问题", "user_id": "user_123", "context": { "topic": "互联网运营", "previous_messages": [] } }

请求参数：

message (string, required) - 用户消息
user_id (string, required) - 用户ID
context (object, optional) - 上下文信息

响应示例：

{ "success": true, "message": "AI分身的回答内容...", "knowledge_used": 3, "confidence": 0.92, "timestamp": "2026-04-01T08:53:00Z" }

2. 知识库查询接口

GET /api/v1/knowledge
GET /api/v1/knowledge?query=产品开发&category=互联网运营&limit=5 Authorization: Bearer YOUR_API_KEY

查询参数：

query (string) - 搜索关键词
category (string) - 知识分类
limit (number) - 返回数量（默认5）

响应示例：

{ "success": true, "results": [ { "id": "001", "category": "互联网运营", "topic": "用户增长", "content": "...", "keywords": ["增长", "用户", "运营"], "relevance": 0.95 } ], "total": 12 }

3. 知识库更新接口

POST /api/v1/knowledge/update
POST /api/v1/knowledge/update Content-Type: application/json Authorization: Bearer YOUR_API_KEY { "id": "001", "category": "互联网运营", "topic": "用户增长", "content": "新的知识内容...", "keywords": ["增长", "用户", "运营"], "examples": ["例子1", "例子2"] }

4. 健康检查接口

GET /api/v1/health
GET /api/v1/health

响应示例：

{ "status": "ok", "uptime": 3600, "model": "kimi-2.5", "knowledge_base_size": 156, "timestamp": "2026-04-01T08:53:00Z" }

🔐 认证方式

所有API请求都需要在Header中包含API Key：

Authorization: Bearer sk-sp-3dcb5dafa03845fda978a7733dcee8c2

📊 错误处理

状态码 说明 示例
200 请求成功 {"success": true, ...}
400 请求参数错误 {"error": "Missing required field: message"}
401 认证失败 {"error": "Invalid API Key"}
500 服务器错误 {"error": "Internal server error"}

💡 使用示例

JavaScript/Node.js

const axios = require('axios'); const response = await axios.post('http://localhost:3000/api/v1/chat', { message: '如何做好小程序冷启动？', user_id: 'user_123' }, { headers: { 'Authorization': 'Bearer sk-sp-3dcb5dafa03845fda978a7733dcee8c2' } }); console.log(response.data.message);

Python

import requests response = requests.post( 'http://localhost:3000/api/v1/chat', json={ 'message': '如何做好小程序冷启动？', 'user_id': 'user_123' }, headers={ 'Authorization': 'Bearer sk-sp-3dcb5dafa03845fda978a7733dcee8c2' } ) print(response.json()['message'])
